# Nature Forward Vector Asset Package

This folder contains vector-ready assets converted from the provided menu image.

## Best-use files

- `nature_forward_full_spread_vector_trace.svg` - full side-by-side menu spread, traced as pure SVG paths.
- `nature_forward_brunch_menu_vector_trace.svg` - left/brunch menu page, pure SVG paths.
- `nature_forward_dinner_menu_vector_trace.svg` - right/dinner menu page, pure SVG paths.
- `nature_forward_logo_vector_trace.svg` - transparent-background trace of the original logo area.
- `nature_forward_full_spread_vector_trace.pdf` - print-ready vector PDF of the full spread.
- `nature_forward_brunch_menu_vector_trace.pdf` - print-ready vector PDF of the brunch page.
- `nature_forward_dinner_menu_vector_trace.pdf` - print-ready vector PDF of the dinner page.
- `nature_forward_editable_menu_template.svg` - editable live-text SVG rebuild/starter layout.
- `nature_forward_logo_rebuilt_editable.svg` - cleaner editable live-text logo rebuild.
- `nature_forward_leaf_mark.svg` - clean vector leaf mark/icon.
- `nature_forward_landing_page_starter.html` - simple responsive landing page starter that references the SVG menu/logo assets.

## Notes

- The trace SVG/PDF files are the most faithful reproduction of the provided image. They are vector paths, not embedded raster images.
- The trace files are scalable and print-ready, but the lettering is converted into outlines/paths, so text is not easily editable.
- The editable SVG template uses live text and vector shapes. It is easier to manage and modify, but it should be proofread and refined before final commercial print.
- The QR code in the traced menu is the original visual trace. The QR-like block in the editable template is a placeholder and should be replaced with a real QR code before publishing.
- For brand-grade logo work, use `nature_forward_logo_rebuilt_editable.svg` or rebuild the logo with the exact brand fonts if available.
- No SVG file contains embedded raster `<image>` elements. The print PDFs were also checked for embedded raster image objects.

## Suggested workflow

1. Use the trace PDF files for immediate printing.
2. Use the separate SVG page files for web/landing page placement.
3. Use the editable template when changing menu text, prices, layout, or branding.
4. Replace the QR placeholder and confirm all prices/menu copy before publishing.
